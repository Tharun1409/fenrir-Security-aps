# APS Security Dashboard

A frontend dashboard for monitoring and managing security scans.  
This project visualizes scan results, vulnerabilities, and scan progress through an interactive dashboard interface.

APS Security Dashboard is a modern web interface for managing and monitoring automated security scans.

The platform allows users to launch scans, monitor real-time scan progress, view vulnerability findings, and analyze scan logs through an intuitive dashboard.

The UI is designed with a clean cybersecurity-style interface and supports both light and dark themes.
##  Features

- Login / Authentication UI
- Dashboard with Severity Overview
- Scan Table with Status & Progress
- Vulnerability Indicators
- Search Scans by Name
- Filter Scans by Status
- Scan Details Page
- Live Scan Console
- Finding Log Panel
- Responsive Layout
- Dark / Light Theme Support
